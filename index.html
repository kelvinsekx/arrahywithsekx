Definition of array

Array provides us the opportunity to display things in a very easy way without leaving us with the ability to do wonders with it.

In a more layman term you can call array a LIST. Where you can throw items into and separate with a comma. Yea, if I were not to be a programmar, I would have named it list and not ARRAY

It provides YOU and I with a javascript object power over list of items.

Is array an object?? Yea it is. I wont even have to talk too much to convince you. Lets quickly use that rarely utilized method "typeof". try console.log(typeof []). Hhen hhen, what does it print out?? Ekkkkkkkkkkkkkkxactly, it did printed //object

Back to our definition. I was saying array in javascript gives us an object power. Any wonders you can imagine can be exert on array.

It syntax is fun too. [//lolo], it just cant be fun than that. It can take in just almost anything from numbers to string to alphanumeric rubbish and even images. What a wawu

Check this out: ['sola','werepe', 45,'lol']. You can pass them to a named variable too like this var anotherArray =['sola','werepe', 45,'lol'].
